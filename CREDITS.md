## About MathZ

MathZ is a modern, feature-rich calculator application for Android and iOS devices. It combines essential calculation tools with an intuitive user interface designed for everyday use.

### Key Features
- **Basic Calculator Mode** - Standard arithmetic operations
- **Scientific Calculator Mode** - Advanced mathematical functions
- **Unit Converter** - 8 categories with 50+ unit conversions
- **Calculation History** - Track and reuse previous calculations
- **Customizable Settings** - Precision, haptic feedback, and theme options
- **Modern Design** - iOS-native interface with dark mode support

### Technology Stack
- **Framework:** React Native with Expo
- **Language:** TypeScript
- **Styling:** NativeWind (Tailwind CSS)
- **Math Engine:** math.js
- **State Management:** React Context + AsyncStorage
- **Build System:** EAS Build (Expo Application Services)

### Version
**1.0.0** - Initial Release
